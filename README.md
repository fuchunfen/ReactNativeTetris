# ReactNativeTetris
俄罗斯方块ReactNative版本的练手项目，
项目是参考和学习大神的react项目[react-tetris](https://github.com/chvin/react-tetris)而来，第一次见到时十分震惊和羡慕。终于在近期公司要求学习TypeScript的档口，又是年后这个相对清闲一点，于是就想用这个来练手。
